# Система интеллектуальной оптимизации маршрутов

## 📌 Описание

Проект представляет собой систему управления маршрутами, которая минимизирует нагрузку на водителей и автоматически корректирует маршруты на основе внешних факторов, таких как погодные условия, дорожные происшествия и новости.

Система анализирует внешние источники данных и принимает автономные решения о пересчёте маршрутов.

---

## 🏗 Архитектура

Монорепозиторий:

- `backend` — сервер API на NestJS
- `frontend` — клиентское приложение на Next.js

---

## ⚙️ Технологии

### Backend
- NestJS
- PostgreSQL + PostGIS
- Prisma
- Redis (планируется)
- WebSocket
- BullMQ (планируется)
- Python AI сервис (планируется)

### Frontend
- Next.js
- TypeScript
- Tailwind CSS
- TanStack Query
- Leaflet
- Socket.io-client

---

## 🎯 Цели проекта

- Автоматический пересчёт маршрутов
- Анализ рисков на основе погоды и новостей
- Принятие решений с помощью AI
- Минимальное взаимодействие водителя
- Обновления в реальном времени
